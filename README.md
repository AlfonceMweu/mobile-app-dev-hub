# Mobile App Dev Hub

A responsive web blog designed to share tutorials and articles about mobile application development. The platform features posts on technologies such as Flutter, React Native, and Kotlin, helping beginners understand modern mobile development concepts.

## Live Demo
https://final-web-dev-project.vercel.app/

## Features
- Responsive blog-style layout
- Navigation menu with multiple pages
- Articles on:
  - Flutter basics
  - React Native components
  - Kotlin for Android
- Clean and readable UI
- Hosted online using Vercel

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation and Setup
To run the project locally:

1. Clone the repository

2. Navigate to the project folder

3. Open `index.html` in your browser  
or start a local server (recommended).
